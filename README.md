# Linear-Regression-Analysis-Model-Evaluation-on-World-Data

- Implements **Linear Regression** on a global dataset  
<br>

- Performs **data preprocessing** and feature selection  
<br>

- Trains and evaluates the regression model  
<br>

- Uses evaluation metrics such as **R-squared** to assess performance  
<br>

- Identifies limitations of the model due to **low accuracy / poor fit**  
<br>

- Highlights the need for more advanced models  
<br>

- Suggests alternative approaches such as:
  - Decision Tree  
  - Random Forest  
  - Other machine learning models  
<br>

- Demonstrates the importance of **model evaluation and selection** in machine learning
